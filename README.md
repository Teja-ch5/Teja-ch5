## Projects

**[AI Automation & Rapid Prototyping System](https://github.com/Teja-ch5/ai-automation-system)**
Full-stack Flask web app with LangChain, Groq LLaMA, SQLite, and n8n. Three AI modules — Task Decomposition, Data Extraction, and Rapid Prototyping. Every task completion triggers an automatic Gmail notification via n8n webhook.
`Python` `Flask` `LangChain` `Groq` `SQLite` `n8n`

---

**[LLM-Powered Multi-Industry KPI Dashboard](https://github.com/Teja-ch5/Kpi-dashboard)**
AI-powered dashboard that analyzes operational KPIs across 5 industries. AI scores each entity 1-10, identifies issues, and recommends actions. Renders as a dynamic HTML dashboard.
`Python` `Groq` `LLaMA` `HTML` `CSS`

---

**[AI Agent & Automated Workflow](https://github.com/Teja-ch5/ai-agent-workflow)**
AI agent that processes tasks, extracts structured data using LLM, and automatically logs everything to Google Sheets via Zapier webhook.
`Python` `Groq` `LangChain` `Zapier` `Google Sheets`

---

**[Customer Churn Prediction](https://github.com/Teja-ch5/customer-churn-prediction)**
Predictive modeling pipeline on 500K+ records using Logistic Regression, Random Forest, and XGBoost — 87% accuracy. Deployed on AWS S3 and EC2 with automated ETL retraining.
`Python` `Scikit-learn` `XGBoost` `AWS S3` `AWS EC2`

---

**[Accident Detection CNN](https://github.com/Teja-ch5/accident-detection-cnn)**
VGG16-based image classifier trained on CCTV footage to detect road accidents. 90.82% accuracy on binary classification.
`Python` `TensorFlow` `VGG16` `OpenCV`

---

**[Order Tracking System](https://github.com/Teja-ch5/order-tracking-system)**
Full-stack web application for parcel logistics with real-time shipment tracking and role-based access control.
`PHP` `MySQL` `Apache` `Bootstrap` `JavaScript`
