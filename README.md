## Projects

**[Customer Churn Prediction](https://github.com/Teja-ch5/customer-churn-prediction)**
Predictive modeling pipeline on 500K+ records using Logistic Regression, Random Forest, and XGBoost — 87% accuracy. Deployed on AWS S3 and EC2 with automated ETL retraining. Presented churn drivers and retention recommendations to stakeholders.
`Python` `Scikit-learn` `XGBoost` `AWS S3` `AWS EC2`

---

**[Accident Detection CNN](https://github.com/Teja-ch5/accident-detection-cnn)**
VGG16-based image classifier trained on CCTV footage to detect road accidents. Transfer learning on ImageNet weights, achieving 90.82% accuracy on binary classification (Accident / Non-Accident).
`Python` `TensorFlow` `VGG16` `OpenCV` `Google Colab`

---

**[Order Tracking System](https://github.com/Teja-ch5/order-tracking-system)**
Full-stack web application for parcel logistics with real-time shipment tracking, role-based access (admin/staff/customer), and a MySQL database normalized to 3NF. Built with PHP, Apache, Bootstrap, and jQuery.
`PHP` `MySQL` `Apache` `Bootstrap` `JavaScript`

---

**[MERN NoteBoard](https://github.com/Teja-ch5/MERN-NoteBoard)**
Collaborative note-taking board built on the MERN stack with user authentication and real-time updates.
`MongoDB` `Express` `React` `Node.js`

---

**[Produce Filter App](https://github.com/Teja-ch5/produce-filter-app)**

`JavaScript` `HTML` `CSS`
